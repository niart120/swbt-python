# swbt-python ドキュメント

`swbt-python` は、Python から NX 向けの仮想 Bluetooth HID 入力デバイスを扱うためのライブラリです。

## 文書一覧

- [docs/api.md](api.md): 公開 API、インポート、接続 API、入力 API、例外の仕様。
- [docs/usage.md](usage.md): 初回ペアリング、再接続、ボタン、スティック、ニュートラル入力など目的別の利用例。
- [docs/hardware.md](hardware.md): 実機、Bluetooth アダプタ、ドライバー、ペアリングと再接続、トラブルシューティング。
- [docs/agent-brief.md](agent-brief.md): AI エージェントが未実装 API を作らないための短い仕様。
- [docs/release-notes.md](release-notes.md): リリースごとの破壊的変更、移行手順、検証範囲。
